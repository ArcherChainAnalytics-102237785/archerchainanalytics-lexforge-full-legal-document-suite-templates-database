## PART 1: THE OPEN-SOURCE PLAY

### What We Open-Source (Public GitHub)

**Repository:** `ArcherChainAnalytics-Mahihkan/lexforge-open`  
**License:** MIT (permissive—enables building)  
**Audience:** Developers, agencies, enterprises wanting to fork/extend

**Public Components:**

1. **Master Schema (lexforge-schema-v1.json)**
   - Full template metadata structure
   - Variable injection system
   - Compliance tag framework
   - Cross-reference architecture
   - *Why:* Become the de facto standard schema; everyone builds against it

2. **Database Generator (lexforge-db-generator.py)**
   - Complete Python source
   - Build custom legal document databases
   - Compose from templates
   - Full documentation
   - *Why:* Users can build their own databases; we own the tool

3. **Composition API (FastAPI reference)**
   - Document assembly engine
   - Variable injection logic
   - Output rendering (DOCX, PDF, JSON, Markdown)
   - Full source code
   - *Why:* Developers implement their own LexForge clones on our standard

4. **CLI Tools**
   - `lexforge-init` — Initialize new database
   - `lexforge-build` — Compile database from templates
   - `lexforge-validate` — Schema validation
   - `lexforge-serve` — Local API server
   - *Why:* Every developer can build locally; we provide hosting convenience tier

5. **Validator & Testing Framework**
   - Schema validation
   - Compliance tag verification
   - Cross-reference checking
   - Document completeness audit
   - *Why:* Community polices quality; we set the bar

### What We Keep Proprietary (Competitive)

**Closed Components:**

1. **Pre-built Database (1,026 documents)**
   - You can build your own; you can't use ours without licensing
   - Costs us $500K+ to create; costs competitors $5M+ to replicate
   - Commercial moat: We have 3-5 year head start

2. **Institutional Legal Content**
   - 19-language translations (proprietary vendor)
   - Jurisdictional compliance audits (legal counsel)
   - Multi-language legal review (international law firms)
   - *Why:* This is the hard part; takes years to replicate

3. **White-Label Hosting Infrastructure**
   - Managed cloud deployment
   - Custom domains
   - Multi-tenant support
   - HIPAA/SOC2/ISO27001 compliance
   - Dedicated infrastructure
   - *Why:* We host; you don't have to

4. **API Platform & Ecosystem**
   - Managed API (rate limiting, auth, billing)
   - Webhook integrations
   - Zapier/Make.com connectors
   - Marketplace integrations
   - *Why:* Network effects; value grows with integrations

5. **LexForge Certified Program**
   - Badging system for compliant documents
   - Certification for extensions/custom docs
   - Insurance backing for certified suites
   - *Why:* Trust + network effects
